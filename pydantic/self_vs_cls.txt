| Feature     | Instance Method            | Class Method                                |
| ----------- | -------------------------- | ------------------------------------------- |
| Decorator   | None                       | `@classmethod`                              |
| First Arg   | `self` (instance)          | `cls` (class)                               |
| Called On   | An instance                | The class itself                            |
| Purpose     | Act on existing model data | Create or modify model before instantiation |
| Example Use | `user.save_to_db()`        | `User.from_json(json_data)`                 |
